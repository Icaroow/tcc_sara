Olá, para você conseguir usufluir disso, será necessário algumas coisas, segue a lista:
Rust(Via WSL, direto no Windows ou numa VM) | Rode este comando se estiver usando Unix(Mac,Linux e WSL)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
Caso esteja usando Windows, será necessário entrar no site Rust e encontrar a versão

Após isso, rode o seguinte comando:
Cargo build -> cargo run
